+специализация
дочерний класс является подтипом родительского
AWindow - BWWindow

+спецификация (abstract)


-конструирование
дочерний использует методы родителя, но не является подтипом родителя

-обобщение
дочерний обобщает методы родителя для более общего класса
Window - ColoredWindow

+расширение
добавление новых методов строк к классссу родителя

-/+ограничение
запрещение некоторых методов родителя

-/+варьирование
класс наследник является вариантом родителя

+/-комбинирование
несколько классов родителей - только для интерфейсов в java
